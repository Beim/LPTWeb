## Usage

```bash
$ git clone https://github.com/Beim/LPTWeb
$ cd LPTWeb
$ npm install
$ npm start         # visit http://localhost:8000
```

More instruction at [documentation](http://pro.ant.design/docs/getting-started).

## Compatibility

Modern browsers and IE11.
